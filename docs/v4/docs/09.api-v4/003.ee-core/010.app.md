###  介绍
框架入口文件，所有模块的初始化都在此文件进行。

###  使用方法
```javascript
const { ElectronEgg } = require('ee-core');
const app = new ElectronEgg();
```

###  API列表

###  register

注册生命周期事件
```javascript
app.register("before-close", function () {});
```

###  run

启动应用
```javascript
app.run();
```
