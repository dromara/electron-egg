###  介绍
日志模块

###  使用方法
```javascript
const { 
  createLog,
  loadLog,
  logger,
  coreLogger
} = require('ee-core/log');

# 高级语法，占位符
%s：字符串
%d：整数
%i：整数
%f：浮点数
%o：obj对象
%O：obj对象
%c：CSS样式
```

###  API列表
###  createLog(config)

- config [Object] - 日志配置

创建一个自定义的日志实例。

###  logger
获取业务日志实例。
```javascript
logger.info()
logger.error()
logger.debug()
logger.warn()
```

###  coreLogger
获取框架核心日志实例。

```javascript
coreLogger.info()
coreLogger.error()
coreLogger.debug()
coreLogger.warn()
```
