###  介绍
基础app模块

###  使用方法
```go
import (
	"github.com/wallace5303/ee-go/eapp"
)
```
###  API列表
仅做简单介绍。按照go标准，常量/函数/方法等详细信息，都在源码中。

###  func Run() 
运行程序。

###  func Close()
退出程序。

###  func ReadPackage()
读取 package.json 数据。

### func Register()
注册事件。目前只支持beforeClose事件。

```
eapp.Register("beforeClose", func() {
	// 关闭数据库
	sqlitelib.CloseDatabase()
})
```
