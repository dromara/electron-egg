title: SSL Certificate
---

1. Download the certificate generation tool:

URL: [https://github.com/FiloSottile/mkcert/releases](https://github.com/FiloSottile/mkcert/releases)
![ssl-github.png](/img/other/ssl-github.png)

2. Open cmd/PowerShell and navigate to the mkcert tool directory
3. Install

![ssl-1.png](/img/other/ssl-1.png)

4. Generate the certificate

![ssl-2.png](/img/other/ssl-2.png)

5. (Optional) Rename the certificate files

For example: change the `.pem` extension to `.crt`, and the `-key.pem` extension to `.key`
