nvm is a Node.js version management tool.

### Download and Install nvm
- Windows: [Download](https://github.com/coreybutler/nvm-windows/releases )
- Linux/macOS: [Download](https://github.com/nvm-sh/nvm)

### Install Node.js with nvm
Install command:
```bash
# Install a Node.js version
nvm install 14.16.0
```
Use a specific Node.js version: `nvm use <version>`
```bash
nvm use 14.16.0
```
Use the Taobao Node mirror: `nvm node_mirror`
```bash
nvm node_mirror https://npmmirror.com/mirrors/node/
```
Use the Taobao npm mirror: `nvm npm_mirror`
```bash
nvm npm_mirror https://npmmirror.com/mirrors/npm/
```
