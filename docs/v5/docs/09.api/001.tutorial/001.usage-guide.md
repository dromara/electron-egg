###  Introduction
The core functionality of the electron-egg framework is driven by `ee-core` and `ee-bin`. It currently includes **100+** APIs, greatly enhancing the framework's capabilities.

###  Module List

#### ee-core
Framework core functionality, providing APIs

#### ee-bin
Scripts and CLI tools, responsible for framework development, compilation, encryption, packaging, etc.

#### ee-go
Go language support
