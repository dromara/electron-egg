### Introduction
Utility library module.

### Usage
```javascript
const Utils = require('ee-core/utils');
```

### API List
### getPackage()
Get the contents of package.json in the project root directory.

### getMAC(iface)

- iface [String] - If provided, restricts MAC address extraction to this interface

Get a valid MAC address

### isMAC(macAddress)

- macAddress [String] - mac address

Whether it is a valid MAC address

### machineId()

- original [Boolean] - If true return original value of machine id, otherwise return hashed value (sha-256), default: false

(Async) Get machine id, returns a Promise object.

### machineIdSync()

- original [Boolean] - If true return original value of machine id, otherwise return hashed value (sha-256), default: false

(Sync) Get machine id.

### isFileProtocol()
Whether it is a file:// protocol

### isWebProtocol()
Whether it is an http:// | https:// protocol

utils/is module
