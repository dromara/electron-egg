###  Introduction
Storage module - SQLite database.

###  Usage
```javascript
const { SqliteStorage } = require('ee-core/storage');
```
### API List

### SQLite Database
```javascript
const dbFile = path.join(getDataDir(), "db", this.dbname);
const sqliteOptions = {
  timeout: 6000,
  verbose: console.log
}
this.storage = new SqliteStorage(dbFile, sqliteOptions);
this.db = this.storage.db;
```
#### storage.db
```javascript
# Insert data    
const insert = this.db.prepare(`INSERT INTO ${table} (name, age) VALUES (@name, @age)`);
insert.run(data);
```

#### storage.name
Database name

#### storage.mode
Path mode: ":memory:", "absolute", "relative", "onlyName"

#### storage.getDbDir
Get the db file directory

#### storage.fileName
Formatted database name
