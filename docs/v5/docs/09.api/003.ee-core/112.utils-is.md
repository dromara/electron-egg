title: utils/is

### Introduction
Utility library is module.

### Usage
```javascript
const UtilsIs = require('ee-core/utils/is');
```

### API List
### renderer()
Whether it is a renderer process

### main()
Whether it is a main process

### osx()
Whether it is a macOS platform

### macOS()
Whether it is a macOS platform

### windows()
Whether it is a Windows platform

### linux()
Whether it is a Linux platform

### x86()
Whether it is an x86 architecture

### x64()
Whether it is a 64-bit architecture

### sandbox()
Whether it is running in a sandbox (macOS)

### mas()
Whether it is currently running, launched from the macOS App Store.

### windowsStore()
Whether it is currently running, launched from the Windows App Store.

### all()
Check whether all passed arguments (functions) are true.

### none()
Check whether all passed arguments (functions) are false.

### one()
Check whether any of the passed arguments (functions) is true.
