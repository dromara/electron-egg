### Introduction
Process utility module.

### Usage
```javascript
const {
  allEnv,
  env,
  isProd,
} = require('ee-core/ps');
```

### API List
### allEnv()
All environment variables of the current process

### env()
Get the current environment, local for local, prod for production

### isProd()
Whether it is a production environment

### isDev()
Whether it is a development environment

### isRenderer()
Whether it is a renderer process

### isMain()
Whether it is a main process

### isForkedChild()
Whether it is a Node.js child process

### processType()
Current process type: browser, renderer, child

### appName()
Application name

### appVersion()
Application version

### getDataDir()
Get the data directory. dev: project directory/data, prod: user home directory/.appname/data

### getLogDir()
Get the log storage path. dev: project directory/logs, prod: user home directory/.appname/logs

### getBundleDir()
Get the bundled resource directory: project directory/public/electron

### getRootDir()
Get the root directory (dev: project root directory, prod: app user data directory)

### getBaseDir()
Project root directory.

### getElectronDir()
Get the electron directory

### getPublicDir()
Get the public directory

### getExtraResourcesDir()
Get the extra resources directory

### getAppUserDataDir()
Get the application user data directory

### getExecDir()
Get the exec directory, the root directory of the executable (exe, dmg, dep) after APP installation

### getUserHomeDir()
Get the user home directory

### getUserHomeAppDir()
Get the user home/appname directory

### getUserHomeHiddenAppDir()
Get the user home/.appname directory

### getMainPort() Not available yet
Get the main process port

### getSocketPort()
Get the built-in socket port

### getHttpPort()
Get the built-in http port

### isPackaged()
Whether the app is packaged

### exit(code)
- code [Number] - Exit code

Process exit

### exitChildJob(code)
- code [Number] - Exit code

Exit a ChildJob process

### isChildJob()
Task type ChildJob

### isChildPoolJob()
Task type ChildPoolJob

### getArgumentByName()
Get command line arguments
