### Introduction
Utility library helper module.

### Usage
```javascript
const UtilsHelper = require('ee-core/utils/helper');
```

### API List
### fnDebounce()

- fn [Function] - Callback function
- delayTime [Time] - Delay time (ms)
- isImediate [Boolean] - Whether to call immediately
- args [type] - Parameters passed to the callback function

Debounce function.

### getRandomString()
Random 10-character string

### mkdir(filepath)

- filepath [String] - File path

Create a directory

### chmodPath(path, mode)

- path [String] - File path
- mode [String] - Permission; e.g., 755, 777

Modify file permissions

### compareVersion(v1, v2)

- v1 [String] - Version number
- v2 [String] - Version number

Compare version numbers

### stringify(obj, ignore = [])
> ee-core: v2.1.0

- obj [String] - Object
- ignore [Array] - Properties to ignore in the object

Serialize an object to a JSON string

### validValue(value)

- value [String]

Whether the given parameter is a valid value, i.e.: not undefined / null / ''

### fileIsExist(filepath)

- filepath [String]

Check whether a file exists

### getValueFromArgv(argv, key)

- argv [Array]
- key [String]

Format argv array to an object and get the value for the specified key
