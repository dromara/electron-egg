### Introduction
Exception catching module. Enabled in both the main process and ChildJob.

### Usage
```javascript
const {   
  loadException,
  uncaughtExceptionHandler,
  unhandledRejectionHandler,
  uncaughtExceptionMonitorHandler
} = require('ee-core/exception');
```

### API List
### loadException()
Enable exception catching. This simultaneously calls uncaughtExceptionHandler() / unhandledRejectionHandler()
```javascript
loadException();
```

### uncaughtExceptionHandler()
Triggered when an exception is thrown on a process and not caught, and silences the exception.

### unhandledRejectionHandler()
Triggered when a rejected exception in a Promise is not caught using catch in synchronous tasks. This event is also triggered even if catch is used in asynchronous contexts.
