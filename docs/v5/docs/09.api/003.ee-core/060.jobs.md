###  Introduction
Task module.

Note: All Electron-related APIs cannot be used in child processes

###  Usage
```javascript
const { ChildJob, ChildPoolJob } = require('ee-core/jobs');
```

### Third-party Usage
1. Create a job file

Create a **./electron/jobs/example/timer.js** file in the project root directory, [job example code](https://github.com/dromara/electron-egg/blob/demo/electron/jobs/example/timer.js)

2. Usage is the same as above

###  API List
####  ChildJob
Get a child process task class. This feature is based on Node.js's child_process module.
```javascript
const { ChildJob } = require('ee-core/jobs');
```

####  ChildPoolJob
Create a pool of processes that stay resident in memory, executing tasks directly upon receiving them. This feature is based on Node.js's child_process module.
```javascript
const { ChildPoolJob } = require('ee-core/jobs');
```
