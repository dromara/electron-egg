### Introduction
Storage module. Provides SQLite database operations based on better-sqlite3.

::: tip Note
better-sqlite3 is **lazily loaded** — the native .node binding is only imported via dynamic `import()` when `SqliteStorage.init()` is called, not at module import time. Projects that don't use SQLite will never load the native module.
:::

### Usage
```javascript
import { SqliteStorage } from 'ee-core/storage';
// import { SqliteStorage } from 'ee-core';  // also works
```
