###  Introduction
Log module

###  Usage
```javascript
const { 
  createLog,
  loadLog,
  logger,
  coreLogger
} = require('ee-core/log');

# Advanced syntax, placeholders
%s: string
%d: integer
%i: integer
%f: floating point number
%o: obj object
%O: obj object
%c: CSS style
```

###  API List
###  createLog(config)

- config [Object] - Log configuration

Create a custom log instance.

###  logger
Get the business log instance.
```javascript
logger.info()
logger.error()
logger.debug()
logger.warn()
```

###  coreLogger
Get the framework core log instance.

```javascript
coreLogger.info()
coreLogger.error()
coreLogger.debug()
coreLogger.warn()
```
