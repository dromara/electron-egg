### Introduction
Utility library IP module.

### Usage
```javascript
const UtilsIP = require('ee-core/utils/ip');
```

### API List
### publicIpv4(options)

- options [Object]
   - type [String] - Query type: 'http' | 'dns' | 'all'
   - timeout [Int] - Timeout, default 3000 milliseconds

Get public IPv4 address
### publicIpv6(options)

- options [Object]
   - type [String] - Query type: 'http' | 'dns' | 'all'
   - timeout [Int] - Timeout, default 3000 milliseconds

Get public IPv6 address
