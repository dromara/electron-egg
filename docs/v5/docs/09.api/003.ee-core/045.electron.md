###  Introduction
Electron functionality wrapped by the framework, provided as APIs in this module.

Note: All Electron-related APIs cannot be used in child processes

###  Usage
```javascript
const { 
    getMainWindow,
    setCloseAndQuit,
    getCloseAndQuit,
} = require('ee-core/electron');
```
###  API List

###  getMainWindow
Return the main window object. Includes all instance APIs of BrowserWindow

###  setCloseAndQuit
Set a flag to control window exit behavior.

###  getCloseAndQuit
Return Object. Used to control window exit behavior.
