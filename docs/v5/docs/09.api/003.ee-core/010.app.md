###  Introduction
Framework entry file. Initialization of all modules takes place in this file.

###  Usage
```javascript
const { ElectronEgg } = require('ee-core');
const app = new ElectronEgg();
```

###  API List

###  register

Register lifecycle events
```javascript
app.register("before-close", function () {});
```

###  run

Start the application
```javascript
app.run();
```
