### Introduction
Framework entry file. Initialization of all modules takes place in this file. Provides lifecycle event registration and application startup.

### Usage
```javascript
import { ElectronEgg } from 'ee-core';
const app = new ElectronEgg();
```

### API List

### register
Register lifecycle event handlers.

Lifecycle events: `Ready`, `ElectronAppReady`, `WindowReady`, `BeforeClose`, `Preload`.

```javascript
import { ElectronEgg, BeforeClose } from 'ee-core';
const electronEgg = new ElectronEgg();
electronEgg.register(BeforeClose, async () => {
  // cleanup before app closes
});
```

### run
Start the application synchronously (CJS controller loading).

```javascript
await electronEgg.run();
```

### runAsync
Start the application asynchronously (ESM controller loading, uses dynamic `import()`).

```javascript
await electronEgg.runAsync();
```

### EventBus

The `eventBus` singleton manages lifecycle and custom events.

```javascript
import { eventBus } from 'ee-core/app';

// Custom events
eventBus.on('myEvent', (data) => { /* handle */ });
eventBus.emit('myEvent', { key: 'value' });
```
