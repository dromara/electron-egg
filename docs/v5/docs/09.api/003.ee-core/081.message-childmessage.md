### Introduction
Message module. Send messages between the main process and child processes.

### Usage
```javascript
const { childMessage } = require('ee-core/message');
```
### API List
### send(eventName, params = {}, receiver)

- eventName [] - The event name to listen to
- params [] - Parameters
- receiver [] - Receiver, values: 'job' | 'task', default: 'task'

Send a message to the main process. Events listened by task.on().
```javascript
const timerTask = this.myJob.exec('./jobs/example/timer', {jobId});
timerTask.emitter.on('job-timer-progress', (data) => {
  // ...
})

// Send
childMessage.send(eventName, {jobId, number:0, pid:0});
```
### sendToMain(eventName, params = {})

- eventName [] - The event name to listen to
- params [] - Parameters

Send a message to the main process, events listened by myjob.on().
```javascript
# Listen in main process task
myjob.on('job-timer-progress', (data) => {
  // some things
})

# Send in child process
let eventName = 'job-timer-progress';
childMessage.sendToMain(eventName, {jobId, number});
```
