###  Introduction
Electron window functionality wrapped by the framework, provided as APIs in this module.

Note: All Electron-related APIs cannot be used in child processes

###  Usage
```javascript
const {
    getMainWindow,
    restoreMainWindow,
    setCloseAndQuit,
    getCloseAndQuit
} = require('ee-core/electron/window');
```
###  API List
###  getMainWindow()
Get the created main window object.

###  createMainWindow()
Create the Electron main window. Used by ee-core, developers can ignore this.

###  restoreMainWindow()
Restore the main window.

###  setCloseAndQuit
Set a flag to control window exit behavior.

###  getCloseAndQuit
Return Object. Used to control window exit behavior.
