### Introduction
Utility library port module.

### Usage
```javascript
const { getPort } = require('../utils/port');
```

### API List
### getPort(options)
- options [Object]
   - port [number | array] - (Optional) Port or iterable of ports to use

Get one or more available ports, returns a Promise object.
```javascript
// Get a random port
const port = await GetPort();

// Get a specified port; if occupied, get a random one
const port = await GetPort({port: 3001});
```
