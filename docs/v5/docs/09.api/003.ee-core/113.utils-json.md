### Introduction
Utility library Json module.

### Usage
```javascript
const UtilsJson = require('ee-core/utils/json');
```
### API List
### strictParse(str)

- str [String] - String

Convert a string to an object
### readSync(filepath)
- filepath [String] - Absolute path of the file

Synchronously read a JSON file.
### writeSync(filepath, str, options)

- filepath [String] - Absolute path of the JSON file
- str [String|Object] - Object or string object
- options
  - space [Number] - Indentation, default 2	

Synchronously write an object or object string to a file.

### read(filepath)
Asynchronously read a JSON file. Returns a promise.

### write(filepath, str, options)

- filepath [String] - Absolute path of the JSON file
- str [String|Object] - Object or string object
- options
  - space [Number] - Indentation, default 2

Asynchronously write an object or object string to a file.
