###  Introduction
Cross-language service module.

###  Usage
```javascript
const { cross, Cross } = require('ee-core/cross');
```

### API List
### run(service, opt = {})
- service [String] - Service name, typically the language being used
- opt [Object] - Parameters
    - name [String] - Program process name
    - cmd [String] - Executable program path, or a local executable command
    - directory [String] - Program directory, e.g., the directory where the jar file is located
    - args [Array] - Executable program arguments
    - appExit [Boolean] - Whether to exit the Electron application when the program exits

Create a service (child) process, returns a cross process object.

### kill(pid)
- pid [Int] - Process ID

Kill a process.

### killByName(name)
- name [String] - Program process name

Kill a process by name.

### killAll()
Kill all processes.

### killByName(name)
- name [String] - Program process name

Kill a process by name.

### getUrl(name)
- name [String] - Program process name

Program service address.

### getProcByName(name)
- name [String] - Program process name

Get a process object by name.

### getProc(pid)
- pid [Int] - Process ID

Get a process object by pid.

### getPids()
Get all process pids, returns an array.

### crossProcess


### entity.pid
Process pid.

### entity.name
Process name.

### entity.config
Process configuration, the option object used when creating the process.

### entity.child
The process object spawned via the [child_process](https://nodejs.org/docs/latest-v16.x/api/child_process.html) module, [spawn](https://nodejs.org/docs/latest-v16.x/api/child_process.html#child_processspawncommand-args-options).

### entity.port
Only has a value if the child process service uses a port.

### entity.emitter
An instantiated [EventEmitter](https://nodejs.org/docs/latest-v16.x/api/events.html) module object.

- Method list:
```javascript
emitter.addListener(eventName, listener)
emitter.emit(eventName[, ...args])
emitter.eventNames()
emitter.getMaxListeners()
emitter.listenerCount(eventName)
emitter.listeners(eventName)
emitter.off(eventName, listener)
emitter.on(eventName, listener)
emitter.once(eventName, listener)
emitter.prependListener(eventName, listener)
emitter.prependOnceListener(eventName, listener)
emitter.removeAllListeners([eventName])
emitter.removeListener(eventName, listener)
emitter.setMaxListeners(n)
emitter.rawListeners(eventName)
emitter[Symbol.for('nodejs.rejection')](err, eventName[, ...args])
```

### entity.kill(timeout)
- name [int] - Delay retry time

Kill the process.

### entity.getUrl()
Program service address.

### entity.getArgsObj()
Return opt.args as an object
