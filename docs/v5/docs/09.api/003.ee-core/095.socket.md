### Introduction
Communication service module. Provides three transport modes:

- **SocketServer**: Socket.IO server for third-party software communication (Go, Python, etc.)
- **HttpServer**: Koa-based RESTful HTTP server
- **IpcServer**: Electron IPC server for main ↔ renderer process communication

All three share the `resolveControllerFn()` routing mechanism. Channel pattern: `controller/{name}/{method}`.

### Usage
```javascript
import {
  SocketServer,
  HttpServer,
  IpcServer,
  loadSocket,
  createSocketServer,
  createHttpServer,
  createIpcServer,
  getSocketServer,
  getHttpServer,
  getIpcServer,
  Koa,
  IoServer,
  IoClient,
} from 'ee-core/socket';
```

### API List

### getSocketServer()
Get the SocketServer instance (**non-null**). Throws Error if called before `Ready` lifecycle event.

```javascript
const SocketServer = getSocketServer();
SocketServer.io.emit("channel-name", "data");
```

::: tip Note
Changed from v5: `getSocketServer()` returns `SocketServer` (not `SocketServer | null`). Throws Error if called before services are created.
:::

#### SocketServer.io
The instantiated Socket.IO server object.

- Properties: [https://socket.io/docs/v4/server-api/#attributes](https://socket.io/docs/v4/server-api/#attributes)
  - `io.engine`
  - `io.sockets`
- Methods: [https://socket.io/docs/v4/server-api/#methods](https://socket.io/docs/v4/server-api/#methods)
  - `io.emit(eventName[, ...args])`
  - `io.on(eventName, listener)`
  - `io.close([callback])`
  - ...and more

#### SocketServer.connect()
Create a `connection` and listen to channel, enabling communication between renderer process and main process controller.

#### SocketServer.socket
The socket object established through `io.on('connection')`.
Docs: [https://socket.io/docs/v4/server-api#socket](https://socket.io/docs/v4/server-api#socket)

### getHttpServer()
Get the HttpServer instance (**non-null**). Throws Error if called before `Ready` lifecycle event.

```javascript
const HttpServer = getHttpServer();
const httpApp = HttpServer.getHttpApp();
```

::: tip Note
Changed from v5: `getHttpServer()` returns `HttpServer` (not `HttpServer | null`).
:::

### getIpcServer()
Get the IpcServer instance (**non-null**). Throws Error if called before `Ready` lifecycle event.

```javascript
const IpcServer = getIpcServer();
```

::: tip Note
Changed from v5: `getIpcServer()` returns `IpcServer` (not `IpcServer | null`).
:::

### Koa
The original Koa object (`require('koa')`). Provided as a base library.

```javascript
import { Koa } from 'ee-core/socket';
```

### IoServer
The original socket.io server object (`require('socket.io')`). Provided as a base library.

```javascript
import { IoServer } from 'ee-core/socket';
```

### IoClient
The original socket.io-client object (`require('socket.io-client')`). Provided as a base library.

```javascript
import { IoClient } from 'ee-core/socket';
```
