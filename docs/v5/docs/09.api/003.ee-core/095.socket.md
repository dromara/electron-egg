###  Introduction
Provides socket communication functionality.

- Can use socket.io-client or Node.js's net module to connect to remote (Java, Go, etc.) socket services
- Can connect to the framework's socket service in third-party projects to communicate with the EE framework

###  Usage
```javascript
const {
  Koa,
  IoServer,
  IoClient,
  loadSocket,
  getSocketServer,
  getHttpServer,
  getIpcServer
} = require('ee-core/socket');
```

###  API List
###  getSocketServer()
Get the socketServer instance object.
```javascript
const { getSocketServer } = require('ee-core/socket');

// Send messages to frontend io-client on() events or other language socket listening events
const SocketServer = getSocketServer();
SocketServer.io.emit("client event name to listen", "data");
```

#### SocketServer.io
The instantiated server object, containing the following properties and methods:

- Properties
```javascript
// Docs: https://socket.io/docs/v4/server-api/#attributes
io.engine
io.sockets
```

- Methods
```javascript
// Docs: https://socket.io/docs/v4/server-api/#methods
io.adapter([value])
io.attach(httpServer[, options])
io.attach(port[, options])
io.attachApp(app[, options])
io.bind(engine)
io.close([callback])
io.disconnectSockets([close])
io.emit(eventName[, ...args])
io.emitWithAck(eventName[, ...args])
io.except(rooms)
io.fetchSockets()
io.in(room)
io.listen(httpServer[, options])
io.listen(port[, options])
io.of(nsp)
io.on(eventName, listener)
io.onconnection(socket)
io.path([value])
io.serveClient([value])
io.serverSideEmit(eventName[, ...args][, ack])
io.serverSideEmitWithAck(eventName[, ...args])
io.socketsJoin(rooms)
io.socketsLeave(rooms)
io.timeout(value)
io.to(room)
io.use(fn)
```

#### SocketServer.connec()

Create a `connection` and listen to channel `c1`, enabling communication between the renderer process and the main process controller.

#### SocketServer.socket

The `socket` object established through SocketServer.io('connection'),
Docs: `https://socket.io/docs/v4/server-api#socket`

You can call the properties/methods on this object to extend functionality.

###  getHttpServer()
Get the httpServer instance object.
```javascript
// Import module at the top
const { getHttpServer } = require('ee-core/socket');
```

###  getIpcServer()
Get the ipcServer instance object.
```javascript
// Import module at the top
const { getIpcServer } = require('ee-core/socket');
```

###  Koa
The original Koa object, equals require('koa'). Provides a base library - avoiding users having to install it repeatedly
```javascript
const { Koa } = require('ee-core/socket');
```

###  IoServer
The original socket.io object, equals require('socket.io'). Provides a base library - avoiding users having to install it repeatedly
```javascript
const { IoServer } = require('ee-core/socket');
```

###  IoClient
The original socket.io-client object, equals require('socket.io-client'). Provides a base library - avoiding users having to install it repeatedly
```javascript
const { IoClient } = require('ee-core/socket');
```
