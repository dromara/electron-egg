###  Introduction
Configuration module

###  Usage
```go
import (
	"github.com/wallace5303/ee-go/econfig"
)
```

###  API List
Brief introduction only. Following Go conventions, detailed information for constants/functions/methods is in the source code.

###  func Init()
Initialize configuration, used by ee-go.

###  func Get(key string) any
Get the value of a specific property.

###  func GetAll() map[string]any
Get all property values.

###  func GetLogger() map[string]any
Get the logger property values.

###  func GetCoreLogger() map[string]any
Get the core logger property values.

###  func GetHttp() map[string]any
Get the http property values.

###  func GetStatic() map[string]any
Get the static resources property values.
