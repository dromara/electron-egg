###  Introduction
Base app module

###  Usage
```go
import (
	"github.com/wallace5303/ee-go/eapp"
)
```
###  API List
Brief introduction only. Following Go conventions, detailed information for constants/functions/methods is in the source code.

###  func Run() 
Run the program.

###  func Close()
Exit the program.

###  func ReadPackage()
Read package.json data.

### func Register()
Register events. Currently only supports the beforeClose event.

```
eapp.Register("beforeClose", func() {
	// Close database
	sqlitelib.CloseDatabase()
})
```
