## Introduction

操作系统工具模块，提供平台检测、OS 信息获取与用户目录查找功能。基于 `runtime.GOOS` 判断当前操作系统，通过 `gopsutil/host` 获取详细的平台信息，支持跨平台（Windows、Linux、macOS）的用户主目录查找。

## Import

```go
import "github.com/wallace5303/ee-go/eos"
```

## API

### IsWindows()

**描述**: 判断当前运行环境是否为 Windows 系统。通过 `runtime.GOOS` 检测，返回 `"windows"` 时为 `true`。

**参数**: 无

**返回值**: `bool` — `true` 表示当前系统为 Windows，`false` 表示不是

**示例**:

```go
if eos.IsWindows() {
    fmt.Println("当前运行在 Windows 系统")
}
```

### IsLinux()

**描述**: 判断当前运行环境是否为 Linux 系统。通过 `runtime.GOOS` 检测，返回 `"linux"` 时为 `true`。

**参数**: 无

**返回值**: `bool` — `true` 表示当前系统为 Linux，`false` 表示不是

**示例**:

```go
if eos.IsLinux() {
    fmt.Println("当前运行在 Linux 系统")
}
```

### IsMacOS()

**描述**: 判断当前运行环境是否为 macOS 系统。通过 `runtime.GOOS` 检测，返回 `"darwin"` 时为 `true`。功能与 `IsDarwin()` 完全相同。

**参数**: 无

**返回值**: `bool` — `true` 表示当前系统为 macOS，`false` 表示不是

**示例**:

```go
if eos.IsMacOS() {
    fmt.Println("当前运行在 macOS 系统")
}
```

**参见**: IsDarwin()

### IsDarwin()

**描述**: 判断当前运行环境是否为 Darwin（macOS）系统。通过 `runtime.GOOS` 检测，返回 `"darwin"` 时为 `true`。功能与 `IsMacOS()` 完全相同，提供 Darwin 内核名称视角的别名。

**参数**: 无

**返回值**: `bool` — `true` 表示当前系统为 Darwin/macOS，`false` 表示不是

**示例**:

```go
if eos.IsDarwin() {
    fmt.Println("当前运行在 Darwin 系统")
}
```

**参见**: IsMacOS()

### OSPlatform()

**描述**: 获取当前操作系统的详细平台名称。通过 `gopsutil/host.PlatformInformation()` 获取，如 `"ubuntu"`、`"windows"`、`"darwin"` 等。获取失败时返回 `"Unknown"`。

**参数**: 无

**返回值**: `string` — 操作系统平台名称字符串，失败时为 `"Unknown"`

**示例**:

```go
platform := eos.OSPlatform()
fmt.Printf("当前平台: %s\n", platform)
// 输出: 当前平台: ubuntu  或  当前平台: darwin  或  当前平台: windows
```

### GetUserHomeDir()

**描述**: 获取当前用户的主目录路径。优先通过 `user.Current()` 获取；跨平台交叉编译时，Windows 通过 `HOMEDRIVE` + `HOMEPATH` 或 `USERPROFILE` 环境变量获取，Unix 通过 `HOME` 环境变量或 shell 命令获取。

**参数**: 无

**返回值**:

| 返回值 | 类型 | 说明 |
|--------|------|------|
| homeDir | `string` | 用户主目录绝对路径 |
| err | `error` | 获取失败时的错误信息；成功时为 `nil` |

**示例**:

```go
homeDir, err := eos.GetUserHomeDir()
if err != nil {
    fmt.Printf("获取用户目录失败: %s\n", err)
    return
}
fmt.Printf("用户主目录: %s\n", homeDir)
// 输出: 用户主目录: /home/user  或  用户主目录: C:\Users\user
```

**参见**: eruntime.UserHomeDir
