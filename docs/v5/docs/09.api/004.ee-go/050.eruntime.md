## Introduction

运行时环境模块，提供应用运行时的全局状态变量和工具函数。包括版本号、环境标识、应用名称、平台类型、目录路径、HTTP 服务状态等全局信息，以及环境判断和路径获取函数。

## Import

```go
import "github.com/wallace5303/ee-go/eruntime"
```

## API

### Version

**描述**: 框架版本号，默认值为 `"0.1.0"`。

**类型**: `string`

### ENV

**描述**: 运行环境标识，取值为 `"dev"` 或 `"prod"`，默认值为 `"dev"`。

**类型**: `string`

### AppName

**描述**: 应用名称，默认为空字符串，由业务代码在启动时赋值。

**类型**: `string`

### Platform

**描述**: 客户端平台类型，取值为 `"pc"`、`"mobile"` 或 `"web"`，默认值为 `"pc"`。

**类型**: `string`

### IsExiting

**描述**: 标识应用是否正在退出，默认值为 `false`。任务模块在添加或执行任务时会检查此标志，若为 `true` 则跳过任务操作。

**类型**: `bool`

### Debug

**描述**: 是否启用调试模式，默认值为 `false`。

**类型**: `bool`

### BaseDir

**描述**: 应用根目录路径，初始化时通过 `os.Getwd()` 获取当前工作目录。

**类型**: `string`

### PublicDir

**描述**: 静态资源目录路径，由 `InitDir()` 设置为 `BaseDir + "/public"`。

**类型**: `string`

### UserHomeDir

**描述**: 操作系统用户主目录路径。

**类型**: `string`

### UserHomeConfDir

**描述**: 操作系统用户主目录下的配置文件目录路径。

**类型**: `string`

### UserHomeAppDir

**描述**: 操作系统用户主目录下的应用数据目录路径。

**类型**: `string`

### WorkDir

**描述**: 应用工作目录路径。

**类型**: `string`

### DataDir

**描述**: 应用数据存储目录路径。

**类型**: `string`

### TmpDir

**描述**: 应用临时数据存储目录路径。

**类型**: `string`

### Port

**描述**: HTTP 服务监听端口，默认值为 `"0"`（表示未设置）。

**类型**: `string`

### SSL

**描述**: 是否启用 SSL，默认值为 `false`。

**类型**: `bool`

### HttpServerIsRunning

**描述**: HTTP 服务器是否正在运行，默认值为 `false`。

**类型**: `bool`

### InitDir()

**描述**: 初始化 PublicDir 目录路径，将其设置为 `filepath.Join(BaseDir, "public")`。应在应用启动早期调用。

**返回值**: 无返回值

**示例**:

```go
eruntime.InitDir()
fmt.Println(eruntime.PublicDir) // 输出: /path/to/app/public
```

### IsProd()

**描述**: 判断当前是否为生产环境，即 `ENV == "prod"`。

**返回值**: `bool` — 若为生产环境返回 `true`，否则返回 `false`

**示例**:

```go
if eruntime.IsProd() {
    fmt.Println("running in production mode")
}
```

### IsDev()

**描述**: 判断当前是否为开发环境，即 `ENV == "dev"`。

**返回值**: `bool` — 若为开发环境返回 `true`，否则返回 `false`

**示例**:

```go
if eruntime.IsDev() {
    fmt.Println("running in development mode")
}
```

### IsDebug()

**描述**: 判断是否启用了调试模式，即 `Debug` 的值。

**返回值**: `bool` — 若启用调试返回 `true`，否则返回 `false`

**示例**:

```go
if eruntime.IsDebug() {
    fmt.Println("debug mode is enabled")
}
```

### Pwd()

**描述**: 获取当前可执行文件所在目录的绝对路径。通过 `exec.LookPath` 定位可执行文件，再通过 `filepath.Abs` 获取绝对路径，最终返回其所在目录。

**返回值**: `string` — 可执行文件所在目录的绝对路径

**示例**:

```go
dir := eruntime.Pwd()
fmt.Println("executable directory:", dir)
```
