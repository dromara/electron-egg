title: 从v4升级v5
---

#### 升级指南

0. 准备，建议开发者新开一个项目分支，来做此次升级。
1. 升级依赖包版本：npm i ee-bin@latest -D  && npm i ee-core@latest
2. 调整配置文件：
    - 参照 [demo配置](https://github.com/dromara/electron-egg/tree/master/cmd)
3. 调整 package.json 配置，参照 [package.json](https://github.com/dromara/electron-egg/blob/master/package.json)
4. 删除 js,ts 文件中这种字符串标识：`'[class ExampleController]'`。 v5 版本已经不需要了。


#### 其它说明
新版本的开发流程，推荐使用以下命令。

```bash
# 开发，可在 cmd/bin.js 中启用热重载
npm run dev

# 构建
npm run build

# 预览
npm run start

# 打包
npm run build-xxx
```
