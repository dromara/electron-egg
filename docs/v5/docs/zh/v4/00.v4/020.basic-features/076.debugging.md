### 开发者工具栏

### debug-主进程

#### 

debuger模式下，框架会加载 `./electron` 源码，直接在源码中打断点即可

目前仅支持 主进程为 js 语言的 debug 调试。调试模式下，会动态修改 `package.json` 文件中，main 字段属性值，不要担心，build构建时，会将修改后的值还原。

#### js 语言

1. 升级 `ee-bin@4.1.6` & 升级 `ee-core@4.1.1`

修改配置：`./cmd/bin.js`

```json
electron: {
    args: ['.', '--env=local', '--debuger=true'],
}
```

2. 使用VSCode调试，菜单栏点击运行->添加配置->Node.js

修改配置：`.vscode/launch.json`

```json
{
  "version": "1.0.0",
  "configurations": [
    {
      "name": "ee-debugger",
      "type": "node",
      "request": "launch",
      "cwd": "${workspaceFolder}",
      "runtimeExecutable": "npm",
      "runtimeArgs": [
        "run",
        "dev"
      ],
      "windows": {
        "runtimeExecutable": "npm",
        "runtimeArgs": [
          "run",
          "dev"
        ],
      }
    }
  ]
}
```
