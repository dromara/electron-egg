Plugin features can freely extend the framework's capabilities while keeping the core framework stable and lightweight.

Plugins are divided into two types: `free plugins` and `paid plugins`.

> Although the framework has hundreds of thousands of users, there is almost no revenue source. We hope to offer some additional plugin features or services to cover server costs.
