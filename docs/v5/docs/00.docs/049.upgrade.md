---
title: Upgrading from v4 to v5
---

#### Upgrade Guide

0. Preparation: It is recommended that developers create a new project branch for this upgrade.
1. Upgrade dependency package versions: npm i ee-bin@latest -D  && npm i ee-core@latest
2. Adjust configuration files:
    - Refer to [demo configuration](https://github.com/dromara/electron-egg/tree/master/cmd)
3. Adjust package.json configuration, refer to [package.json](https://github.com/dromara/electron-egg/blob/master/package.json)
4. Remove string identifiers like `'[class ExampleController]'` from js/ts files. These are no longer needed in v5.


#### Additional Notes
For the new version's development workflow, it is recommended to use the following commands.

```bash
# Development, hot-reload can be enabled in cmd/bin.js
npm run dev

# Build
npm run build

# Preview
npm run start

# Package
npm run build-xxx
```
