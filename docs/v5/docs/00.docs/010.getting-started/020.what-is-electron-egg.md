### One-Line Introduction

electron-egg is a `business framework`.

Just like Spring for Java, ThinkPHP for PHP, Nuxt.js for Vue...

Electron provides basic functions and APIs, but when you build a project, business logic and code engineering must be implemented yourself. `electron-egg` provides this engineering capability.
