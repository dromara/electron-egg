macOS高版本（12），移除了默认Python

1. brew安装
```bash
brew install python@3.8
```
2. 编辑 ~/.zshrc
```bash
vim ~/.zshrc

# python
alias python='/usr/local/bin/python3'
export PATH="/usr/local/opt/python@3.8/libexec/bin:$PATH"
```

3. 使修改生效
```
source ~/.zshrc
```
