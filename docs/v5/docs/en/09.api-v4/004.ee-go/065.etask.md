###  介绍
任务队列模块

###  使用方法
```go
import (
	"github.com/wallace5303/ee-go/eapp"
)
```
###  API列表
任务队列可以更好的处理异步任务。

###  func AddTask(action string, handler interface{}, args ...interface{}) 
添加任务。

###  func AddTaskWithTimeout(action string, handler interface{}, timeout time.Duration, args ...interface{})
添加超时任务。

###  func Status()
任务状态。

### func ExecTask()
执行任务。

### func Every()
每隔一段时间执行任务。
