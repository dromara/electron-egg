## 什么是 electron-egg

electron-egg 是一款**入门简单、跨平台、企业级桌面软件开发框架**，基于 Electron 封装，提供完整的模块化架构、通信机制与构建工具链。无论是内部工具、数据看板还是商用客户端，electron-egg 都能帮助你快速交付桌面应用。

项目由两个核心包驱动：

| 包名 | 定位 | 说明 |
|------|------|------|
| **ee-core** | 运行框架 | TypeScript 双格式（CJS + ESM）输出，负责应用生命周期、配置加载、控制器管理、通信服务、子进程任务、日志与存储 |
| **ee-bin** | 构建工具 | CLI 命令行工具，基于 esbuild + Commander，提供开发启动、bundle 打包、代码加密、图标生成、增量更新等功能 |

两个包均为独立 npm 包，生产环境通过 `npm install ee-core ee-bin` 安装，当前 monorepo 结构仅用于本地开发便利。

## 核心特性

<Steps>
<Step title="跨平台支持">
一套代码构建 Windows、macOS（含 ARM64）、Linux 三平台安装包，无需平台特定适配。
</Step>

<Step title="模块化架构">
ee-core 采用 Boot → loadConfig → loadLog → loadDir → loadController → loadSocket → loadElectron 的生命周期管线，每个环节独立可扩展。
</Step>

<Step title="前端无关">
内置 Vue 3 + Ant Design 示例前端，但框架本身不绑定任何前端技术栈。你可以替换为 React、Svelte 或纯 HTML。
</Step>

<Step title="多通道通信">
主进程与渲染进程之间支持三种通信方式：

- **IPC**：`controller/{name}/{method}` 通道模式，轻量高效
- **HTTP Server**：内置 HTTP 服务，支持 RESTful 接口调用
- **Socket.IO**：实时双向通信，适合推送与长连接场景
</Step>

<Step title="Go/Python 后端支持">
可选集成 Go（gin）或 Python（FastAPI）后端服务，通过 HTTP/Socket 与 Electron 主进程通信，适合计算密集型或 AI 相关业务。
</Step>

<Step title="子进程任务池">
内置 ChildJob（单进程）、ChildPoolJob（进程池）与 LoadBalancer（8 种负载均衡算法），支持 CPU 密集型任务分发与进程复用。
</Step>

<Step title="代码加密">
通过 ee-bin 的 encrypt 命令，支持 V8 字节码编译与 JavaScript 混淆，保护商业代码安全。
</Step>
</Steps>

## 适用场景

| 场景 | 说明 |
|------|------|
| 企业内部工具 | 运维面板、CRM 客户端、审批系统等 |
| 数据可视化 | 实时看板、监控仪表盘、报表工具 |
| 商用客户端 | 桌面端 SaaS 产品、离线优先应用 |
| 跨平台工具 | 需同时覆盖 Win/Mac/Linux 的工具类应用 |
| AI/计算密集型 | 搭配 Go/Python 后端处理模型推理或大数据计算 |

## 与其他 Electron 框架对比

| 特性 | electron-egg | electron-vite | Electron Forge |
|------|:---:|:---:|:---:|
| 中文文档与社区 | 原生中文 | 部分 | 英文为主 |
| Go/Python 后端集成 | 内置支持 | 无 | 无 |
| 子进程任务池 | 内置 | 无 | 无 |
| 代码加密（字节码） | 内置 | 无 | 无 |
| IPC/HTTP/SocketIO 通信 | 三种方式 | IPC | IPC |
| 构建工具 | esbuild（ee-bin） | Vite | Electron Forge |
| 前端绑定 | 无绑定 | Vue/React/Svelte | 无绑定 |
| 配置分层 | config.default → config.local/config.prod | Vite 配置 | Forge 配置 |

<Note>
对比仅供参考，各框架定位与设计目标不同。electron-egg 更侧重**全栈桌面应用**场景（前端 + 主进程 + 后端服务 + 任务池），而非纯前端构建优化。
</Note>

## 技术栈概览

- **Electron** — 桌面应用容器
- **TypeScript** — ee-core 与 ee-bin 均为 TypeScript 实现，输出双 CJS + ESM 格式
- **esbuild** — 构建打包引擎（ee-bin）
- **Vue 3 + Ant Design** — 示例前端（可替换）
- **better-sqlite3** — 内置 SQLite 存储
- **Pino** — 高性能日志（支持 pino-roll 滚动与 pino-pretty 格式化）
- **pnpm workspace** — monorepo 包管理
