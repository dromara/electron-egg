## 🍬 Features

- **Cross-platform**: One codebase, packaged for Windows, Mac, Linux, UOS, Deepin, Kylin and more.
- **Architecture**: Single business process / modular / multi-task, making large projects simple.
- **Simple & efficient**: Supports js and ts.
- **Frontend agnostic**: Works with any frontend tech — vue, react, html, vite, etc.
- **Engineering**: Build desktop software with frontend & server-side development mindsets.
- **High performance**: Event-driven, non-blocking IO.
- **Feature-rich**: Config, IPC, jobs, plugins, database, updates, packaging, tools and more.
- **Secure**: Supports bytecode encryption and minify-obfuscation encryption.
- **Extensible**: Extend business logic with other languages — c/c++, python, go, java, etc.
